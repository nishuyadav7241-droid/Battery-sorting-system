Why this exists

Problem & Solution

Battery recycling is broken. This project fixes the sorting step.

⚠️

The Problem

Used batteries — lithium-ion, alkaline, lead-acid, coin cells — are mixed together and sorted manually in recycling facilities. Manual sorting is slow, inconsistent, exposes workers to hazardous materials, and leads to improper disposal that damages the environment.

✅

The Solution

A conveyor-based AI sorting system that uses a camera to identify the battery type in real time, then automatically routes it to the correct bin using servo-controlled gates — no human needed at any step.

System flow

How it works

Eight steps from battery placement to correct bin — fully automated.

1

Battery placed on conveyor

Arduino starts the conveyor motor at fixed speed (PWM 150) automatically on power-on.

2

Camera detects the battery

YOLOv8 object detection model spots the battery in the live webcam feed and crops the region.

3

AI classifies the type

EfficientNet-B0 classifier identifies one of six types: 18650, 21700, 9V, CoinCell, DCell, or LeadAcid.

4

Python sends type to ESP32

Battery type is sent over USB serial as a text string. ESP32 stores it in a queue (up to 20 batteries).

5

IR sensor detects battery arrival

Arduino detects the battery at the sorting position, stops the conveyor, and sends READY to ESP32.

6

ESP32 calculates timing

Based on the battery type, ESP32 calculates the exact travel time needed to reach the correct servo gate.

7

Correct servo fires

The servo at the right bin position rotates 90°, pushes the battery in, then returns to home position.

8

Conveyor restarts for next battery

ESP32 sends NEXT to Arduino. Conveyor restarts and the cycle repeats for the next battery in queue.
